About Split-Wiser

Smart bill splitting made simple

Features

  • Splitwise Integration
  • Manual Bill Entry
  • GST Calculation
  • Discount Management
  • Multi-Person Items

How It Works

Connect your Splitwise account or enter names manually
Add individual amounts or multi-person items
Apply GST and discounts automatically
View calculated amounts for each person

About the Developer

Split-Wiser is developed with modern web technologies to make bill splitting easier and more accurate. Built with Angular and PrimeNG for a smooth user experience.
